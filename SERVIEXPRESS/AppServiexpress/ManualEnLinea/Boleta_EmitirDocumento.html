Módulo Boletas / Facturas

Emitir Documento

Esta ventana le permitirá generar boletas o facturas, ya sea para pagar una orden de trabajo o vender un producto o servicio sin orden de trabajo.

Para emitir una boleta, debe primeramente indicar “Cliente”, “Sucursal”, “Empleado” y “Tipo Documento”. A continuación, debe ingresar ítems al documento, ya sean ordenes de trabajo, servicios o productos.

Para agregar una “Orden de Trabajo”, debe haber seleccionado una sucursal y un cliente. Una vez seleccionados, presione el botón” Agregar O.T.”, se desplegará una ventana que le permitirá buscar una orden de trabajo. Una vez seleccionada la orden de trabajo, presione “Agregar Orden”, y el documento cargará la orden indicada.

Para agregar “Servicios”, seleccione una sucursal y luego presione el botón “Agregar Servicios”. En “Servicio” se listarán los servicios disponibles en esa sucursal. Seleccione el servicio para visualizar él costo y presione el botón “Agregar Servicios”, el que se añadirá automáticamente como ítem al documento. Una vez que haya seleccionado todos los servicios deseados, puede cerrar la ventana.

Para agregar “Productos”, seleccione una sucursal y luego presione el botón “Agregar Productos”. En “Producto” se listarán los productos disponibles en esa sucursal. Pude filtrar productos, para obtener una lista específica. Seleccione el producto para visualizar él costo y stock, ingrese la cantidad y presione el botón “Agregar Productos”, el que se añadirá automáticamente como ítem al documento. Una vez que haya seleccionado todos los productos deseados, puede cerrar la ventana.

Si desea quitar ítems del documento seleccione los ítems en la tabla y a continuación presione el botón “Quitar Ítem”.

Una vez completados los datos, seleccione la moneda de pago y presione el botón “Emitir Documento” para generar la venta y almacenar el registro en la base de datos. Además, se le descargará un documento con la información ingresada del documento en formato PDF, dentro de la carpeta “Mis Documentos”.
